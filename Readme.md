A simple command-line tool written in Go to set your Windows desktop wallpaper from a local image, a remote URL, or a random image from a saved directory.


## Features
- Set wallpaper via **URL**
- Set wallpaper from **local file path**
- Set a **random wallpaper** from a saved folder
- Uses native **(user32.dll)**
